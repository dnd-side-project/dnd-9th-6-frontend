# Classcope

**✨온라인 클래스 탐색 여정을 밝혀주는 Classcope✨**

Classcope는 다양한 온라인 클래스 플랫폼을 하나로 통합하여 편리한 검색 기능과 강의 후기 공유 기능을 제공하는 서비스입니다.
